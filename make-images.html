Rising Phoenix — Image Generator

Draws the raster image files the site needs (Facebook/Instagram/text link previews require PNG/JPG — SVG won't render there). Click each Download, then drop the files next to index.html. Fonts load from Google, so stay online when generating.

1. Social share image → og-image.jpg (1200×630)

2. Favicon → favicon-32.png (32×32)

3. Apple touch icon → apple-touch-icon.png (180×180)

4. Square profile icon → profile-icon.png (512×512)

For social media profile photos (Instagram / Facebook / Google).

Done?

  1. Save the files into the website folder (next to index.html).
  2. The favicon & social tags are already wired into the page — no code changes needed.
  3. After the site is live, test the preview at opengraph.xyz or Facebook's Sharing Debugger.
